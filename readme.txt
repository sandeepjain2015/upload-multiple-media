== Description ==
Upload multiple media by Api.
In default Wordpress only provide api to upload only one image by rest api . So when we use from front end by react ,angular or ios or android app for upload multiple image so by this plugin you can upload media to WordPress. 

== Installation ==
THANK YOU FOR YOUR INTEREST IN Upload multiple media by Api.
MINIMUM REQUIREMENTS.
WordPress 3.4+
PHP 5.x
MySQL 5.x
PERFORM A NEW INSTALLATION
After downloading the ZIP file of the plugin,

Log in to the administrator panel.
Go to Plugins Add > New > Upload.
Click “Choose file” (“Browse”) and select the downloaded zip file of the plugin.
Click “Install Now” button.
Click “Activate Plugin” button for activating Upload multiple media by Api

== Frequently Asked Questions ==
= What will be file name for upload multiple media files =
mmbyapi_file_upload[]
= What will be rest api path =
/wp-json/mmbyapi/v1/upload-multiple-images
